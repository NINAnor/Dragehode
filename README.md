# Dragehode overvåking

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.10677920.svg)](https://doi.org/10.5281/zenodo.10677920)

Quarto book to present code and outcomes from the Dragehode monitoring. The live book can be found [here](https://ninanor.github.io/Dragehode/)

To clone the repository please use the following R code:

```
install.packages("remotes")
remotes::install_github("NINAnor/Dragehode")

``` 

This will produce a local replica of the repository on your computer.
